Formularios en HTML

Los formularios en HTML nos permiten recoger información del usuario mediante campos interactivos. Son esenciales para funciones como registros, búsquedas o envíos de datos a un servidor.

La estructura básica de un formulario utiliza la etiqueta <form>, y dentro de esta podemos colocar diferentes tipos de elementos para entrada de datos.

Ejemplo básico







<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">

    <label for="correo">Correo:</label>
    <input type="email" id="correo" name="correo">

    <input type="submit" value="Enviar">
</form>

Botones

Existen distintos tipos de botones para controlar el comportamiento del formulario:

<input type="submit" value="Enviar">
<input type="reset" value="Limpiar">
<button type="button">Personalizado</button>

También se pueden incluir etiquetas como <select> para listas desplegables o <textarea> para textos largos.





<select>
    <option>México</option>
    <option>Colombia</option>
    <option>Argentina</option>
</select>

<textarea>Escribe aquí...</textarea>
