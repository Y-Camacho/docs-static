Listas

Las listas en HTML nos permiten crear conjuntos de elementos en forma de lista dentro de una página web.

Listas desordenadas

  • Arroz
  • Pasta
  • Manzanas
  • Sal
  • Pollo
<ul>
    <li>Arroz</li>
    <li>Pasta</li>
    <li>Manzanas</li>
    <li>Sal</li>
    <li>Pollo</li>
</ul>

Se puede cambiar el tipo de viñeta con el atributo type.

  • Arroz
  • Pasta

  • Arroz
  • Pasta
<ul type="square">
    <li>Arroz</li>
    <li>Pasta</li>
</ul>

<ul type="circle">
    <li>Arroz</li>
    <li>Pasta</li>
</ul>

Listas ordenadas

  1. Juan
  2. Pepe
  3. María
  4. Julia
  5. Pedro
<ol>
    <li>Juan</li>
    <li>Pepe</li>
    <li>María</li>
    <li>Julia</li>
    <li>Pedro</li>
</ol>

También se puede cambiar el tipo de numeración con el atributo type.

  1. Juan
  2. Pepe

  1. Juan
  2. Pepe
<ol type="I">
    <li>Juan</li>
    <li>Pepe</li>
</ol>

<ol type="a">
    <li>Juan</li>
    <li>Pepe</li>
</ol>

También se puede revertir el orden o definir desde dónde empieza la lista.

  1. Juan
  2. Pepe

  1. Juan
  2. Pepe
<ol type="I" reversed>
    <li>Juan</li>
    <li>Pepe</li>
</ol>

<ol start="5">
    <li>Juan</li>
    <li>Pepe</li>
</ol>
