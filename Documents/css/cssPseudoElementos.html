Pseudo elementos

Nos permiten aplicar estilos a una arte específica de un elemento. Para utilizarlos se usa dos dobles puntos (::). Los pseudoelementos aplican a cualquier elemento de bloque.

Ejemplo de aplicación


/* El sigiuente estilo hará que la primera letra del párrafo
    sea más grande */
    .p-speudoelement::first-letter {
        font-size: 32px;
    }

Clasificación de los pseudo elementos

TIPO PSEUDOELEMENTO DESCRIPCIÓN
Contenido generado ::before, ::after Información generada desde CSS, sin existir en el HTML.
Contenido tipográfico ::first-line, ::first-letter Pseudoelementos relacionados con temas de tipografías.
Contenido destacado ::selection, ::target-text, ::spelling-error, ::grammar-error Pseudoelementos para remarcar o destacar información.
WebComponents ::part, ::slotted Pseudoelementos relacionados con WebComponents
View Transition API ::view-transition, ::view-transition-group, ::view-transition-image-pair ::view-transition-new, ::view-transition-old Pseudoelementos de transición de cambio de página
Otros pseudoelementos ::marker, ::placeholder, ::file-selector-button Pseudoelementos de otras categorías variadas

Más pseudoelementos

PSEUDOELEMENTO DESCRIPCIÓN
::marker Aplica estilos a las marcas o símbolos de cada ítem de una lista.
::backdrop Aplica estilos al fondo exterior de un elemento en primer plano (sin que afecte a este).
::placeholder Aplica estilos a los textos de sugerencia de los campos <input>.
::file-selector-button Aplica estilos a los botones de campo <input> de subir archivos.
