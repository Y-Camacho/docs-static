Pseudoclases

Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.

Las pseudo clases aplicarán estilos dependiendo del contexto, la posición o de ciertos estados que tenga el elemento HTML en cuestión. Las pseudoclases comienzan con dos puntos (:).

Ejemplo de aplicación


    /* El siguiente estilo aplica un color verde al fondo 
    del botón cuando el cursor está sobre él (:hover) */
        .botton-aceptar:hover {
            background-color: green;
        }

Clasificación de las pseudoclases

TIPO PSEUDOCLASE DESCRIPCIÓN
Interacción :hover, :active, :focus, :focus-within, :focus-visible Pseudoclases relacionadas con acciones de usuario.
Ubicación :any-link, :link, :visited, :target Pseudoclases relacionadas con enlaces o ubicaciones.
Idioma :lang(), :dir() Pseudoclases relacionadas con idiomas.
Estructura :root, :host, :defined, :empty :first-child, :last-child, :only-child :first-of-type, :last-of-type, :only-of-type :nth-child(), :nth-last-child(), :nth-of-type(), :nth-last-of-type() Pseudoclases de estructura de documentos HTML
Formulario :checked, :indeterminate :enabled, :disabled, :read-only, :read-write, :placeholder-shown, :default :required, :optional, :valid, :invalid, :user-valid, :user-invalid :in-range, :out-of-range Pseudoclases de formularios HTML
Estado :fullscreen, :modal Pseudoclases relacionadas con el estado de modales o similares.
Paginado :first, :left, :right, :blank Pseudoclases de paginado de documentos HTML
