El posicionamiento

Las propiedades FLOAT y POSITION nos permiten modificar la posición natural de cualquier elemento del documento HTML. El navegador posiciona cada elemento teniendo en cuenta el orden en que que aparece al documento (de izquierda a derecha y de arriba a abajo) y el tipo de visualización dependiendo si es un elemento en línea o en bloque. Con FLOAT y POSITION podemos modificar este comportamiento

Tipos de posicionamiento

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky

Para mover y posicionar los elementos (con relative, absolute, fixed o sticky) utilizaremos las propiedades:

  • top (vertical, eje y)
  • bottom (vertical, eje y)
  • left (horizontal, eje x)
  • right (horizontal, eje x)
  • z-index (profundidad, eje z)

Estas 5 propiedades NO funcionan con el valor por defecto static! Porque static significa respetar el flujo original del posicionamiento de los elementos.

Las propiedades top y left tiene prioridad respecto a botton y right respectivamente. En caso de conflicto entre aplicar top o botton, se aplicará top. Lo mismo pasa con left y right.

Posicionamento static

En este posicionamiento no se aplican left, right, top, booton ni z-index

Posicionamento reletive

En este posicionamiento el elemento conserva las dimenciones y el posicionamento original dentro del flujo de documento. Si se mueve toma como referencia su posición original.

Posicionamento absolute

El elemente pieder sus dimenciones (width y heigth) y posición original al flujo del documento. Si es mueve pudede tomar como referencia dos elementos:

  • El primer ancestro con posicionamiento relativo.
  • Si no encuentra ningún ancentro relativo, se mueve respectoa al documento HTML.

Posicionamiento FIXED

Pierde las dimenciones y posición original del flujo del documento. Si se mueve toma como referencia el documento HTML y queda fija en la posición donde se ha movido. Cuando se haga un scroll el elemento no se moverá, se quedará fijo.

Posicionamiento Sticky o pegajoso

Es una combinación de posicionamiento relative y fixed. En este tipo de posicionamiento las propiedades top, left, bottom y right no mueven el elemento. En vez de desplazar sirven como punto de referencia mientras el elemento no llega a los valores establecidos. Mientras no llegue a las posiciones establecidas por las propiedades , el elemento se comporta relativamente y cuando llegua a estas posiciones queda fija.

Hay dos opciones para implementarlo

  • El elemento sticky tiene que se hijo directo del body
  • Su elemento contenedor debe tener dimenciones definidas y solamente será sticky dentro de este contenedor.
