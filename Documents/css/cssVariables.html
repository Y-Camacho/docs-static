Variables

Las propiedades personalizadas (a veces denominadas variables CSS o variables en cascada) son entidades definidas por los autores de CSS que representan valores específicos que se reutilizarán en todo un documento.

Los sitios web complejos tienen una gran cantidad de CSS, y esto suele dar como resultado una gran cantidad de valores CSS repetidos. Por ejemplo, es común ver el mismo color usado en cientos de lugares diferentes en las hojas de estilo. Cambiar un color que se ha duplicado en muchos lugares requiere una búsqueda y reemplazo en todas las reglas y archivos CSS. Las propiedades personalizadas permiten definir un valor en un lugar y luego hacer referencia a él en varios otros lugares para que sea más fácil trabajar con él. Otro beneficio es la legibilidad y la semántica. Por ejemplo, --main-text-colores es más fácil de entender que el color hexadecimal #00ff00, especialmente si el color se usa en diferentes contextos.

Cómo crear y usar una variable en CSS

/* Creación de las variables */
            :root {
                --main-text-colores: #00ff00;
                --color-primario: #dd32da;
                --font-size-small: 16px;
            }
        
        /* Utilizar una variable */
            body {
                background-color: var(--color-primario);
            }

En la creación de una variable se hace uso de la pseudoclase :root para declarar todas las variables necesarias. Mientras que para poder utilizar una como valor en un parámetro se debe usar la función var().
