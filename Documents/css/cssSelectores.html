Los selectores se utilizan para delimitar los elementos HTML de una página web a los que se quieren aplicar estilo.Son patrones de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS.

Selectores básicos

/* Selectores de etiqueta: cualquier etiqueta HTML */
p {
    color: orange;
}

/* Selectores identificadores (id, "#"). */
#paragraf-dos {
    color: black
}

/* Selectores de clase (class, ".") */
.paragraf-dos {
    color: green;
}

<body>
    <p id="paragraf-dos" class="paragraf-dos"></p>
</body>

Selectores avanzados


/* Hijos directos ( > ) */
    .fills-directes > li{
        background-color: chocolate;
    }
    
/* Hijos descedientes ( " " ) */
    .fills-descendents b {
        background-color: lightgreen;
    }
    
/* Hermanos generales ( ~ ) */
    .germans-general ~ li {
        background-color: blue;
    }
    
/* Hermanos adyacentes ( + ) */
    .german-adyacent + li {
        background-color: blue;
    }

/* Selectores de atributo (selector[atributo]) */
    .selectors-atributs a[href] {
        background-color: yellow;
    }
    
/* Selectores da tributo con valores concretos (selector[atributo=""]) */
    .selectors-atributs a[target="_blank"] {
        background-color: rgb(122, 62, 207);
    }
    
/* Selector de atributo con un patrón (selector[atributo*=""]) */
    .selectors-atributs a[href*="index"] {
        font-size: 24px;
    }
    
/* Selector de atributo que conmience con un patrón (selector[atributo^=""]) */
    .selectors-atributs a[href^="#"] {
        text-decoration: none;
    }
    
/* Selector de atributo que acabe con un patrón (selector[atributo$=""]) */
    .selectors-atributs a[href$="-css"] {
        font-family: Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif;
    }
    
/* Selector de atributo que encuentre un patrón en una lista de valores */ 
    (selector[atributo~=""])
    .selectors-atributs a[class~="cursos"] {
        font-style: italic;
    }
    
/* Selector de atributo que encuentre un patrón en una lista de valores */
    y acontinuación del patrón haya un guiòn (-)
    .selectors-atributs a[class|="menu"] {
        color:red;
    }

Selector universal

El selector universal se indica con un asterisco ( * ) y selecciona todos los elementos del documento (o del elemento padre si está encadenado con otro elemento y un operador de combinación descendente, por ejemplo).

Suele utilizarce para cambiar el box model de un documento o para inicializar los valores por defecto que los navegadores asignan a los elementos HTML. 


/* Selector universal ( * ) */
    * {
        box-sizing: border-box;
        margin: 5px;
    }

/* Selector universal contextual */
    .selector-universal-contextual * {
        font-size: 24px;
    }

Agrupación de selectores

Sirven para cuando queremos aplicar los mismos estilos con distintos selectores. utilizaremos la coma (,) para listar los selectores a los que queremos aplicar los mismo estilos. 


    h2, h2, .tittle {
        color: red;
    }
