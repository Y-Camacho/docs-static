Cascading Style Sheets

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación y el diseño de un documento escrito en HTML o XML. Su desarrollo ha evolucionado desde los años 90 para convertirse en una parte esencial del diseño web moderno. A continuación, se detalla una breve historia:

1. Nacimiento de CSS (1994-1996)

  • 1994: Håkon Wium Lie propuso por primera vez la idea de CSS mientras trabajaba en CERN, el laboratorio donde se creó la web. Su propuesta buscaba separar la estructura del contenido (HTML) de la presentación.
  • 1996: El W3C (World Wide Web Consortium) publicó el CSS1, la primera versión oficial. Permitía un control básico del diseño como fuentes, colores y márgenes, pero tenía un soporte limitado en navegadores.

2. CSS2 y el auge de los navegadores (1998)

  • CSS2, publicado en 1998, amplió considerablemente las capacidades, introduciendo:
    • Diseño para dispositivos específicos (pantallas, impresoras).
    • Soporte para layouts como posicionamiento absoluto y relativo.
    • Selectores más avanzados.
  • Durante este tiempo, los navegadores como Internet Explorer y Netscape comenzaron a implementar CSS, pero el soporte era inconsistente.

3. CSS3 y la revolución del diseño web (2000s)

  • El desarrollo de CSS3 comenzó en la década de 2000, pero no fue un solo lanzamiento. En cambio, se dividió en módulos como selectores, diseño de cajas flexibles y animaciones, para facilitar su adopción.
  • Introdujo características como:
    • Media Queries para diseño responsivo.
    • Efectos visuales como sombras, gradientes y transiciones.
    • Nuevas unidades como rem y soporte para tipografías web.
  • Su adopción fue impulsada por la mejora en los navegadores modernos (Chrome, Firefox, Safari).

4. CSS4 y el presente

  • Aunque no existe un "CSS4" oficial, las actualizaciones de CSS continúan a través de módulos. Algunas de las funcionalidades modernas incluyen:
    • Grid Layout y Flexbox para crear diseños complejos y responsivos.
    • Soporte para animaciones avanzadas con @keyframes.
    • Nuevas herramientas como variables CSS (--nombre-var) y funciones avanzadas.

Importancia actual

CSS es esencial en el diseño web, ya que permite personalizar la experiencia del usuario en diferentes dispositivos y navegadores. Su evolución refleja el crecimiento y la madurez de la web como una plataforma dinámica y estética.
