Estilos

Los estilos nos permiten cambiar la apariencia de los distintos elementos de un documento HTML. Para hacer uso de ellos hay que modificar las propiedades de los elementos, y esto se puede hacer de distintas formas.

Estilos en línea

Se aplican directamente a un elemento con la propiedad style. 

<!--Estilos en linea -->
      <h1 style="background-color: red;">Introducción al css</h1>

Estilos internos

Se aplican usando la etiqueta style en el head. 

<head>
          <style>
              body {
                  color: red;
              }
          </style>
      </head>

Estilos externos

Se desarrollan en un documento CSS aparte y se enlaza con el HTML a través de la etiqueta link. 

/* Codigo CSS */
      * {
          box-sizing: border-box;
      }
      
      body {
          color: red;
      }
      
<!-- Codigo HTML */ -->
      <head>
          <link rel="stylesheet" href="style.css">
      </head>
