Cascading Style Sheets

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación y el diseño de un documento escrito en HTML o XML. Su desarrollo ha evolucionado desde los años 90 para convertirse en una parte esencial del diseño web moderno. A continuación, se detalla una breve historia:

1. Nacimiento de CSS (1994-1996)

  • 1994: Håkon Wium Lie propuso CSS en CERN para separar contenido y presentación.
  • 1996: El W3C publicó CSS1 con funciones básicas como fuentes, colores y márgenes.

2. CSS2 y el auge de los navegadores (1998)

  • CSS2 amplió capacidades con soporte para:
    • Diseño para dispositivos específicos.
    • Posicionamiento absoluto y relativo.
    • Selectores más avanzados.
  • Internet Explorer y Netscape comenzaron a adoptar CSS, con soporte irregular.

3. CSS3 y la revolución del diseño web (2000s)

  • CSS3 se dividió en módulos para facilitar su adopción.
  • Introdujo:
    • Media Queries para diseño responsivo.
    • Sombras, gradientes y transiciones.
    • Unidades como rem y tipografías web.
  • Navegadores modernos impulsaron su adopción.

4. CSS4 y el presente

  • Aunque no existe un "CSS4", se siguen lanzando mejoras mediante módulos:
    • Grid Layout y Flexbox.
    • Animaciones con @keyframes.
    • Variables CSS (--nombre-var) y funciones modernas.

Importancia actual

CSS es esencial en el diseño web, ya que permite personalizar la experiencia del usuario en diferentes dispositivos y navegadores. Su evolución refleja el crecimiento y la madurez de la web como una plataforma dinámica y estética.
